#  **Question 3**

This question uses **Node.js**, **npm packages**, and a **.env file**.

### Steps:
1. Go to the folder:

cd Question3
```
2. Install dependencies:
npm install
```

3. Create a `.env` file (not included in GitHub):

Example:
EMAIL=your_email_here
PASSWORD=your_email_app_password
```

4. Run:
node index.js
```
