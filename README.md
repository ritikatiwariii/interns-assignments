# Igniters Hub Internship Assignment (Node.js)

This repository contains my completed solutions for the Igniters Hub Internship Assignment. The assignment includes four questions, each solved in its own folder.

Folder Structure:

```
interns-assignments/
│
├── Question1/
│     └── index.js
│
├── Question2/
│     ├── index.js
│     ├── input.txt
│     └── output.txt
│
├── Question3/
│     ├── index.js
│     ├── package.json
│     ├── package-lock.json
│     ├── .env
│     └── node_modules/
│
└── Question4/
      └── index.js
```

## How to Run Each Question

### Question 1
1. Go to the Question1 folder:  
   ```bash
   cd Question1
   ```
2. Run the solution:  
   ```bash
   node index.js
   ```

### Question 2
1. Go to the Question2 folder:  
   ```bash
   cd Question2
   ```
2. Run the solution:  
   ```bash
   node index.js
   ```

### Question 3
1. Go to the Question3 folder:  
   ```bash
   cd Question3
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Add your `.env` file with the following content:  
   ```
   EMAIL_USER=your_email_here
   EMAIL_PASS=your_email_password_here
   ```
4. Run the solution:  
   ```bash
   node index.js
   ```

### Question 4
1. Go to the Question4 folder:  
   ```bash
   cd Question4
   ```
2. Run the solution:  
   ```bash
   node index.js
   ```
